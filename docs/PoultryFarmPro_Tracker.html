⚠️ Critical Constraints

  • operationType is on Pen model, not PenSection
  • penWorkerAssignment HAS isActive — dashboard: no filter; generate/tasks: filter isActive:true
  • Seed IDs are slugs — use z.string().min(1) not .uuid()
  • Egg formula: (crates×30)+loose+cracked
  • Feed formula: (bags×bagWt)+(bagWt−remaining) only if remaining > 0.1
  • @db.Date ranges: always use Date.UTC(y,m,d)
  • WeightSample model REMOVED — use WeightRecord
  • All hooks before any early returns (React rules)
  • Next.js 16 dynamic routes: const params = await rawParams
  • Schema changes: SQL → prisma db pull → prisma generate — never edit schema.prisma directly
  • All API routes must scope every query with tenantId
